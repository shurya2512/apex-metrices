# Phase 6 Context: Frontend Telemetry Charting & Timing Tower

## AI Agent Instructions & Objectives
**Goal:** Build the core visualization tools: a live Timing Tower and the Head-to-Head Telemetry charts.

**Why this matters:** This is the flagship feature of ApexMetrics. Engineers and fans must be able to visually compare the braking, acceleration, and shifting habits of two different drivers on the same lap.

## Success Criteria
1. A user can select two drivers from the Timing Tower.
2. The Telemetry overlay renders multi-line charts comparing Speed, RPM, Throttle, Brake, and Gear for both drivers.
3. Hovering over a specific timestamp on one chart (e.g., Speed) simultaneously highlights that exact timestamp on the other charts (e.g., Brake and Gear).
